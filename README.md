# MMGCN: multi-modal multi-view graph convolutional networks for cancer prognosis prediction

# Dependencies
MMGCN is built with Python 3.7 with the following packages:
* torch == 1.13.1
* snfpy == 0.2.2
* pandas == 1.3.5
* numpy == 1.21.6
* scikit-learn == 1.0.2

# Usage
We provide the scripts for running MMGCN.  


`python MMGCN.py`
